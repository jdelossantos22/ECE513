Full control at your fingertips Control the thermostat from anywhere

Smart Home lets the user set the thermostat temperature from the web. The user can set the mode off, heat, or cool. The user can set the fans on or auto. The thermostat uses DHT11 to sense the current temperature in the house

Full RGB Customization Control the smart bulb anywhere

Smart Home lets the user control the brightness of the smart bulb. The user can set the brightness on or set it to auto. The user can set a timer for the light to be on or off, and customize the hue of the light. The IOT device uses a photoresistor to sense the brightness and set the brightness for the smart bulb

Security and Peace of Mind Get alerts of door open

Smart Home will send alerts to the user whenever the door is open. It will continually send alerts in a timer until the door is closed. The IOT device uses a photo resistor to simulate a door close or open. If the sensor senses brightness above a threshold, it means the door is open and sends an alert to the user

In-Depth Power Consumption Analysis See for yourself.

Smart Home will show the user their power consumption in the previous 24 hours based on their heating and cooling use. The web application will store the information of the power consumption in a database
